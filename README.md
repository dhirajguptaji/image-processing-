# TIFF Image Processing Tools

This repository contains Python tools for processing TIFF images from experimental image folders.

## File

`tiff_tools.py`

This module provides helper functions to:

- Find TIFF images in a folder
- List TIFF images with image numbers
- Average selected TIFF images
- Subtract a background image
- Average images first and then subtract a background image
